# GMBA Case Studies for Discussion Classes

21 case studies featuring **famous brands** and **high-stakes business decisions**.

## GitHub Pages Site

The Wikipedia-style study site lives in [`docs/`](docs/). It contains:

- [`docs/index.md`](docs/index.md): case-study index
- [`docs/theories.md`](docs/theories.md): business and management theory guide
- [`docs/cases/`](docs/cases/): one Markdown page per case study
- [`docs/reports/`](docs/reports/): PDF and QMD report assets linked from case pages

To publish on GitHub Pages, set **Settings -> Pages -> Build and deployment -> Source** to `Deploy from a branch`, then select the branch and `/docs` folder.

**Selection Criteria**:
- Globally recognized brands
- Real consequences (money lost, people affected, companies failed)
- Strong business dilemmas and strategic debates
- Rich in GMBA theory applications
- Neutral topics suitable for multicultural classrooms
